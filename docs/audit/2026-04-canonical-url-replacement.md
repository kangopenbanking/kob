**Date:** 2026-04-20
**Scope:** Full codebase (frontend, edge functions, email templates, shared helpers, docs)
**Standard cited:** Project Knowledge — *Permanent Public Routes & Canonical Domain*
**Status:** PASS

---

## 1. Objective

Ensure every runtime, email, and developer-facing reference to the legacy preview domain `https://kob.lovable.app` is replaced with the canonical production domain `https://kangopenbanking.com`. The `lovable.app` host must only remain inside historical audit documents (Standing Order 2 — The Ratchet).

## 2. Methodology

1. Recursive grep across all source extensions (`.ts`, `.tsx`, `.js`, `.jsx`, `.json`, `.html`, `.md`, `.yaml`, `.yml`).
2. Exclusions: `node_modules`, `dist`, `.git`, `.lovable`, archived `/docs/audit/*` (historical record).
3. In-place `sed` replacement on every active file.
4. Re-scan for residual occurrences.
